# AIリテラシー認定 β版

サーバー不要で動く、AIリテラシー診断の初期プロトタイプです。

## 実装済み

- スマートフォン対応のランディングページ
- 3段階の認定レベル説明
- 5領域・10問のランダム診断
- 8分タイマー、進捗表示、前問への移動
- 100点換算とレベル判定
- 領域別スコアと改善提案
- 認定番号の自動発行
- 認定証の印刷・PDF保存
- 最終結果のブラウザ内保存

## GitHub Pagesで無料公開する方法

1. このフォルダ内の `index.html`、`styles.css`、`app.js`、`README.md` をGitHubリポジトリ直下へアップロードします。
2. GitHubでリポジトリの `Settings` を開きます。
3. 左側の `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. Branchを `main`、フォルダを `/(root)` にして `Save` を押します。
6. 数分後、同じ画面に公開URLが表示されます。

## ローカル確認

`index.html` をブラウザで開くだけで動作します。より確実に確認する場合は、このフォルダで簡易サーバーを起動します。

```bash
python3 -m http.server 8000
```

その後 `http://localhost:8000` を開きます。
